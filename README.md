# Swiggy Review Sentiment Analysis

## Overview

This project analyzes customer reviews from Swiggy and predicts whether the review sentiment is positive or negative using Natural Language Processing (NLP) and Machine Learning.

## Dataset

The dataset contains customer reviews and their corresponding sentiment labels (positive or negative).

## Technologies Used

* Python
* Pandas
* NumPy
* NLTK
* Scikit-learn

## Project Workflow

1. Load the review dataset
2. Clean and preprocess text data
3. Convert text into numerical features using NLP techniques
4. Train a Machine Learning model
5. Predict sentiment of new reviews

## Model

A machine learning classifier is used to identify the sentiment of customer reviews.

## How to Run

Install libraries:
pip install -r requirements.txt

Run the project:
python train.py

## Result

The model predicts whether a Swiggy review is positive or negative.
